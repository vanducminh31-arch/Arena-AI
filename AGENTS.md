# Arena AI - Agent Guidelines

## 🤖 Role Definitions
- **Agent Architect**: Claude Opus 4.6 - Responsible for system design, file structure, and implementation planning.
- **Agent Developer**: GPT-OSS 120B - Responsible for writing clean, optimized, and modern code (HTML, Vanilla CSS, JS).
- **Agent Reviewer**: Claude Sonnet 4.6 - Responsible for code quality, accessibility, and refactoring.
- **Agent Tester**: Gemini 3 Flash - Responsible for unit testing and quick validation.

## 🎨 Design & Aesthetic Standards
- **Theme**: Premium Dark Mode (Deep grays, obsidian, subtle glassmorphism).
- **Colors**: Primary (Electric Violet / Cyber Blue), Secondary (Slate/Charcoal), Accent (Neon Cyan).
- **Typography**: Modern Sans-Serif (Inter, Outfit, or Poppins).
- **Feel**: Smooth, animated, high-end SaaS (inspired by Linear.app and Stripe).

## 🛠 Tech Stack
- **Foundation**: HTML5, Vanilla JavaScript (ES6+).
- **Styling**: Vanilla CSS (CSS Variables, Flexbox/Grid, Framer-motion-like transitions).
- **Build Tool**: Vite (for fast development and production bundling).

## 📝 Workflow
1. **Architect** creates an `implementation_plan.md`.
2. **Developer** implements the foundation (`index.html`, `index.css`).
3. **Developer** builds components modularly.
4. **Reviewer** performs a quality check and suggests polish.
5. **Tester** verifies functionality and performance.

## 🚀 SEO & Performance
- Semantic HTML tags.
- Meta tags for social sharing.
- Optimized image loading and minimal bundle size.
